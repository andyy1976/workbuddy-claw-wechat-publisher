# 智能双文章生成系统

## 📖 概述

智能双文章生成系统是一个高效的自动化内容创作工具，它能够在每次任务中：
1. **智能选择**最热门主题
2. **同时生成**小红书风格文章（≤1000字）和公众号风格文章
3. **自动发布**两篇文章到微信公众号草稿箱

## 🏗️ 系统架构

### 核心文件
```
dual-article-system/
├── dual_article_generator.js      # 主程序（650+行代码）
├── dual_article_config.json       # 系统配置文件
├── test_dual_article.js           # 完整测试套件
├── automation_config.example.json  # 微信公众号API配置示例
├── 双文章生成系统使用说明.md      # 详细使用文档
└── README.md                      # 本文件
```

### 依赖关系
系统依赖于wechat-publisher-plugin的核心功能，特别是微信公众号API集成。

## 🔧 安装与集成

### 1. 复制文件到项目
所有文件已在 `dual-article-system/` 目录中准备好。

### 2. 配置微信公众号API
编辑 `automation_config.json`（基于示例文件）：
```json
{
  "appId": "你的微信公众号AppID",
  "appSecret": "你的微信公众号AppSecret",
  "thumbMediaId": "封面图素材ID"
}
```

### 3. 安装依赖（如未安装）
```bash
npm install axios
```

## 🚀 使用方法

### 基本执行
```bash
cd dual-article-system
node dual_article_generator.js
```

### 测试验证
```bash
cd dual-article-system
node test_dual_article.js
```

### 完整流程
1. 系统会自动选择今日最热门主题
2. 生成小红书风格文章（1000字以内）
3. 生成公众号风格文章（2000-3000字）
4. 保存文章到本地文件
5. 发布到微信公众号草稿箱

## 📊 核心功能

### 智能主题选择
系统内置5个高热度主题库，基于热度评分算法选择：
- AI人工智能（95/100）
- 区块链技术（88/100）
- 绿色科技（85/100）
- 未来办公（82/100）

### 双文章风格生成
#### 小红书风格（≤1000字）
- 亲切的朋友聊天语气
- 实用性强，提供具体建议
- 包含emoji和话题标签
- 适合年轻用户群体

#### 公众号风格（2000-3000字）
- 专业的行业分析视角
- 结构化的内容组织
- 数据驱动的观点论证
- 适合专业读者群体

### 自动化发布
- 一键发布两篇文章到微信公众号草稿箱
- 完整的发布状态监控
- 详细的执行日志记录
