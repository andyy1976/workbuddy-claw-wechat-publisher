# 微信公众号发布器技能 v2.0

## 核心能力

本插件为 OpenClaw 提供完整的微信公众号内容生产和发布能力：

1. **热点抓取与过滤**：实时获取微博热搜，按用户关键词白名单评分
2. **AI文章生成**：调用 DeepSeek / OpenAI / Moonshot 生成 1500-2000 字深度文章
3. **微信排版**：自动转换为微信公众号标准 HTML，无 Markdown 符号残留
4. **话题标签**：文章末尾自动添加蓝色话题标签，提升曝光
5. **一键发布**：调用微信草稿箱 API 直接发布

## 可用命令

| 命令 | 功能 |
|------|------|
| `/wx-setup` | 初始化配置（首次使用必须先运行） |
| `/wx-publish` | 自动抓取热点 → AI生成文章 → 发布草稿箱 |
| `/wx-hotspot` | 查看今日热点排行，选题参考 |
| `/wx-diary` | 将自定义文字/Markdown 发布为公众号文章 |
| `/wx-validate` | 验证公众号 AppID/AppSecret 是否有效 |

## 配置文件说明

配置文件位于 `config/user-config.json`：

```json
{
  "wechat": {
    "appId": "wx...",
    "appSecret": "...",
    "thumbMediaId": "...",
    "author": "你的名字"
  },
  "keywords": {
    "primary": ["人工智能", "AI", "机器人"],
    "secondary": ["互联网", "通信"],
    "exclude": ["明星", "娱乐"]
  },
  "publish": {
    "targetTime": "07:00",
    "minHeat": 50000,
    "contentStyle": "深度分析型"
  },
  "ai": {
    "provider": "deepseek",
    "model": "deepseek-coder",
    "apiKey": "sk-你的API Key",
    "baseUrl": "https://api.deepseek.com"
  }
}
```

## AI 提供商支持

| 提供商 | provider 值 | 推荐模型 |
|--------|------------|---------|
| DeepSeek（推荐） | `deepseek` | `deepseek-coder` |
| OpenAI | `openai` | `gpt-4o-mini` |
| 月之暗面 | `moonshot` | `moonshot-v1-8k` |

## 文章结构（AI生成）

AI 按以下结构生成深度文章：

1. **开头钩子**：震惊数据或画面，引发好奇（80字以内）
2. **三个信号**：从现象到本质，层层递进
3. **深度解构**：权力/尊严/关系的深层分析
4. **文明视野**：宏观趋势，三个转身
5. **实用建议**：3条可操作建议（蓝绿渐变框）
6. **升华结尾**：金句收尾 + 互动引导

## 话题标签（自动添加）

每篇文章末尾自动添加蓝色话题标签：

```
#公众号创作 #公众号运营 #自媒体新人成长记 #公众号写作心得
```

## 排版规范

- ✅ 无 `#` `##` `###` 标题符号
- ✅ 无 `**` `*` 加粗/斜体符号
- ✅ 无 `---` 分隔线
- ✅ 无 `>` 引用符号
- ✅ 标题：蓝色左边框样式
- ✅ 建议框：蓝绿渐变背景
- ✅ 背景框：橙色边框

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| AI 调用失败 | 自动回退到模板生成，不中断发布 |
| Token 获取失败 | 提示检查 AppID/AppSecret |
| 无匹配热点 | 展示全部热点候选 |
| 网络超时 | 重试一次，再失败则报错退出 |
