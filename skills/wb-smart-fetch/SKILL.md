# wb-smart-fetch — 企业数据采集引擎

> 不是 Browse AI 的"网页变Excel"，而是"企业数据变内容"
> 核心差异化：采集的数据 → 产品库 → 内容 → 发布，形成闭环

## 定位

企业内容数字员工的数据采集层。采集目的不是为了"导出Excel"，
而是为了：
1. **完善产品库** — BOM数据、产品参数、技术规格自动入库
2. **竞品对比** — 同类产品参数横向对比，生成对比内容
3. **内容结合** — 产品数据 + AI → 营销文案、技术博客、客户案例

## 三种采集模式

### A. BOM采集（企业内部数据）
从PLM/MES/ERP系统提取产品结构数据：
- 物料清单（零件号、规格、数量、供应商）
- 产品参数（性能指标、技术规格、认证信息）
- 工艺路线（工序、设备、工时）

输入：PLM系统API / Excel文件 / 数据库视图
输出：产品库JSON + 参数矩阵 + 可直接用于内容生成的结构化数据

### B. 竞品采集（外部市场数据）
从竞品网站/行业平台提取同类产品参数：
- 产品型号、核心参数、价格、卖点
- 用户评价、市场份额数据
- 技术白皮书/解决方案描述

输入：竞品URL列表
输出：竞品参数表 + 对比矩阵 + 可引用的内容素材

### C. 行业信源采集（内容原料）
从行业媒体/公众号/论坛采集热点话题和内容：
- 行业趋势、政策变化、技术突破
- 客户痛点、应用场景、成功案例
- 关键词热度、搜索趋势

输入：信源URL + 话题关键词
输出：热点列表 + 内容素材 + AI评分排序

## 闭环流程

```
采集(BOM/竞品/信源) → 入库(产品库) → 生成(AI写文案) → 发布(CMS/微信/小红书)
                     ↘ 对比(竞品矩阵) ↗
                     ↘ 标签(产品关联) ↗
```

每条采集的数据都会：
- 打上产品标签（关联到具体产品线）
- 孯入产品库（可复用、可查询）
- 作为内容生成的基础素材（"我们的XX产品比竞品YY强在ZZ方面"）

## API 端点

| 方法 | 路径 | 功能 |
|------|------|------|
| POST | `/api/skills/smart-fetch/bom/import` | 导入BOM数据（Excel/API/数据库） |
| POST | `/api/skills/smart-fetch/bom/parse` | 解析BOM，提取产品参数 |
| POST | `/api/skills/smart-fetch/competitor/scan` | 竞品参数采集 |
| POST | `/api/skills/smart-fetch/competitor/compare` | 竞品对比矩阵生成 |
| POST | `/api/skills/smart-fetch/source/collect` | 行业信源采集+AI评分 |
| GET  | `/api/skills/smart-fetch/product/catalog` | 产品库查询 |
| POST | `/api/skills/smart-fetch/content/generate-from-product` | 从产品数据生成内容 |

## 数据模型

### 产品库（核心）

```json
{
  "productId": "PLM-XX-001",
  "name": "智能检测终端 V3",
  "category": "工业检测设备",
  "specs": {
    "检测精度": "0.01mm",
    "工作温度": "-40~85℃",
    "通信协议": "MQTT/OPC UA",
    "认证": "CE/UL"
  },
  "bom": [
    { "partNo": "SENSOR-001", "name": "高精度传感器", "qty": 1, "supplier": "博世" },
    { "partNo": "MCU-002", "name": "边缘计算芯片", "qty": 1, "supplier": "NXP" }
  ],
  "competitors": [
    { "name": "竞品A", "specs": { "检测精度": "0.05mm" }, "priceRange": "¥8-12万" }
  ],
  "contentTags": ["智能制造", "工业检测", "边缘计算"],
  "contentGenerated": 3,
  "lastUpdated": "2026-05-21"
}
```

### 对比矩阵

| 参数 | 我们 | 竞品A | 端品B | 优势说明 |
|------|------|-------|-------|---------|
| 检测精度 | 0.01mm | 0.05mm | 0.02mm | 精度提升5倍 |
| 工作温度 | -40~85℃ | -20~60℃ | -30~70℃ | 工业级宽温 |
| 价格 | ¥6万 | ¥8-12万 | ¥7万 | 性价比最优 |

这种对比矩阵可以直接变成营销文案："精度是竞品5倍，价格仅6万"
