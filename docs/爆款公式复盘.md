## 爆款公式

```
爆款 = 公共性强的选题 × 有钩子的标题 × 轻深度的结构 × 广谱的读者代入感 × 热点窗口期发布
```

---

## 一、选题的"公共性"（决定天花板）

### 什么是公共性？
- 有画面感：机器在沙漠里种树，5秒一棵
- 有国家叙事：北斗是中国自己的导航系统
- 有反差：高科技 vs 传统行业
- 有真实热点：确实上了热搜

### 对比分析

| 选题 | 公共性 | 读者范围 |
|------|:------:|----------|
| 北斗种树 | ⭐⭐⭐⭐⭐ | 科技+农业+生态+爱国 |
| 特斯拉机器人 | ⭐⭐⭐ | 科技爱好者 |
| 悟空发布 | ⭐⭐ | 企业服务用户 |
| GPT-5回信 | ⭐ | 哲学思考者 |

## 二、标题的"钩子"（决定点击率）

### 爆款标题四要素

```
热点 + 具体数字 + 社会证明 + 价值承诺
```

### 爆款标题分析

**北斗原文**：

| 元素 | 内容 |
|------|------|
| 热点 | 热搜第一 |
| 具体数字 | 5秒种一棵树 |
| 社会证明 | 61万人围观 |
| 价值承诺 | 静悄悄的农业革命 |

| 标题 | 问题 |

## 三、文章结构（轻深度）

### 爆款结构模板

```
1. 开头：从热搜切入，讲现象（10%）
2. 三个信号：技术/就业/转型（40%）
3. 深度解构：权力转移/趋势分析（30%）
4. 实用建议：普通人如何应对（15%）
5. 升华结尾：人与未来的关系（5%）
```

### 读者预期

✅ **好的预期**："看完我能了解一个社会热点，还能得到一些启发"

❌ **不好的预期**："我被上了一堂哲学课"

### 公众号读者画像

- 想要：轻思考 + 重收获
- 不想要：太深奥、太烧脑、太文艺

## 六、反面案例：为什么"2026年AI新突破攻略"只有几个阅读？

| 问题 | 分析 |
|------|------|
| 标题太泛 | "2026年AI新突破"公众号不吃这套 |
| 内容太浅 | "技术平民化"全是正确的废话 |
| 没有钩子 | 没有案例、数据、反差 |
| 读者不明确 | 既不是小白入门，也不是专业分析 |

## 七、行动清单

### 选题标准
- [ ] 有画面感的社会热点
- [ ] 有公共性（国家叙事/民生相关）
- [ ] 有反差（高科技vs传统）
- [ ] 真实热搜事件

### 标题公式
```
[热点标签] + [具体数字] + [社会证明] + [价值承诺]
```

示例：
- "热搜第一！北斗导航机器人在沙漠5秒种一棵树：61万人围观的背后，是一场静悄悄的农业革命"
- "3万阅读！特斯拉机器人引爆热议：人类文明的分水岭，普通人该如何应对"

### 结构要求
- 10% 热点引入
- 40% 三个信号/趋势分析
- 30% 深度解构
- 15% 实用建议
- 5% 升华结尾

### 发布时机
- 热点出现后 **24小时内** 发布
- 超过48小时不做跟进

## 八、AI生成器优化

### 选题评分系统

```javascript
function scoreTopic(topic) {
    let score = 0;
    
    // 画面感 (20分)
    if (topic.hasVisual) score += 20;
    
    // 公共性 (30分)
    if (topic.isNationalNarrative) score += 15;
    if (topic.isPublicConcern) score += 15;
    
    // 反差感 (20分)
    if (topic.hasContrast) score += 20;
    
    // 热度 (30分)
    score += topic.heatScore * 0.3;
    
    return score;
}
```
