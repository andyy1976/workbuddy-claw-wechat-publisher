# 公众号文章转发钩子技巧

## 核心原理

> 转发从来不是"求"出来的，而是"设计"出来的

**三类转发动机**：
- 🔹 **共鸣价值**：你说出了他说不出口的话
- 🔹 **实用价值**：他想收藏，或转给需要的人
- 🔹 **话题价值**：适合丢进群里讨论

---

## 三个关键位置

### 📍 25%位置：实用钩子

**时机**：读者刚读完开头，初步信任建立

**技巧**：干货段落后面，加一句"转给谁"

**模板**：
```
[干货内容]

——这份[清单/步骤/指南]，直接转给[目标人群]，[具体好处]。
```

**示例**：
```
服务器报错502，别急着重启。按这个顺序排查：
①看日志时间戳……②检查上游服务……③……

——这份排查清单，直接转给IT同事，下次再报错他能少熬夜。
```

---

### 📍 50%位置：共鸣钩子

**时机**：文章过半，情绪高点

**技巧**：最有共鸣的金句，引导"截图发朋友圈"

**模板**：
```
"[扎心真相/金句]"

——这段话，适合截图发朋友圈。不说矫情，只说懂的人自然懂。
```

**示例**：
```
"中年职场人的崩溃，从来不是被裁员那一刻，而是你发现自己的技能，在招聘网站上已经搜不到职位了。"

——这段话，适合截图发朋友圈。不说矫情，只说懂的人自然懂。
```

---

### 📍 75%位置：话题钩子

**时机**：文章即将收尾，认知铺垫充分

**技巧**：抛出有讨论空间的观点，引导"发到群里讨论"

**模板**：
```
"[有争议/讨论空间的观点]"

——这个观点丢到群里，能聊半小时。你站哪边？顺手转给同事，看看他怎么说。
```

**示例**：
```
"有人说AI会替代文案，也有人说AI只是工具。其实分水岭不是技术，而是——你用AI节省下来的时间，是用来刷短视频，还是用来洞察用户。"

——这个观点丢到群里，能聊半小时。你站哪边？顺手转给同事，看看他怎么说。
```

## AI生成时自动添加

### Prompt模板更新

```
生成文章时，请在以下位置自动添加转发钩子：

1. 文章25%处（约第3-4段末尾）：
   - 如果有干货/清单/步骤，添加实用钩子
   - 格式：——这份XXX，直接转给XXX，能XXX。

2. 文章50%处（约中间位置）：
   - 找到最有共鸣的金句，添加共鸣钩子
   - 格式：——这段话，适合截图发朋友圈。

3. 文章75%处（约倒数第2-3段）：
   - 抛出一个讨论点，添加话题钩子
   - 格式：——这个观点丢到群里，能聊半小时。
```

## 代码集成

```javascript
// 在 article-optimizer.cjs 中添加
function addShareHooks(content) {
    const paragraphs = content.split('\n\n');
    const total = paragraphs.length;
    
    // 25%位置：实用钩子
    const pos25 = Math.floor(total * 0.25);
    // 检测是否有干货内容，添加钩子
    
    // 50%位置：共鸣钩子
    const pos50 = Math.floor(total * 0.5);
    // 找到金句，添加钩子
    
    // 75%位置：话题钩子
    const pos75 = Math.floor(total * 0.75);
    // 添加讨论引导
    
    return paragraphs.join('\n\n');
}
```
