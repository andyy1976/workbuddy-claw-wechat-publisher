## 一、现有功能总结

wechat-publisher-plugin 是一个完整的微信公众号自动化发布工具，主要功能包括：

1. **热点抓取**：支持微博/知乎/B站/小红书四大平台热点抓取
2. **关键词过滤**：通过primary/secondary/exclude三级关键词体系对热点评分过滤
3. **卡兹克风格AI写作**：集成独家写作风格，产出真人味儿的深度文章
4. **markdown转微信HTML**：自动转换排版，去除markdown符号
5. **一键发布草稿箱**：调用微信公众号API直接发布到草稿箱
6. **增强发布系统**：支持从markdown文件发布，自动生成封面图、内容配图

整体架构采用「JavaScript主引擎 + Python增强发布」的混合架构，JavaScript负责热点抓取、AI调用、微信API交互，Python负责图片生成（封面/配图）。

## 三、优化方案

### P0 必须优化

#### 优化P0-01：修复WebAccess路径硬编码
**修改方式**：
```javascript
// 原代码：
const WebAccess = require('C:/Users/tuan_/.openclaw/skills/web-access-wrapper.js');

// 修改为：
let WebAccessPath;
// 优先尝试相对路径（插件内自带）
const localPath = path.join(baseDir, '../web-access-wrapper.js');
const openclawPath = path.join(process.env.HOME || process.env.USERPROFILE, '.openclaw/skills/web-access-wrapper.js');

if (fs.existsSync(localPath)) {
  WebAccessPath = localPath;
} else if (fs.existsSync(openclawPath)) {
  WebAccessPath = openclawPath;
} else {
  console.warn('⚠️  WebAccess模块未找到，热点抓取可能失败');
  WebAccess = null;
}
```
**收益**：适配不同环境，不依赖硬编码绝对路径

#### 优化P0-02：短视频模块改为可选加载
**修改方式**：使用try-catch包裹require，不存在时静默降级
```javascript
let ShortVideoGenerator = null;
try {
  ShortVideoGenerator = require('C:/Users/tuan_/.openclaw/skills/short-video-generator/main.js');
} catch (e) {
  console.warn('⚠️  短视频生成模块未找到，短视频功能不可用');
  ShortVideoGenerator = null;
}
```
**收益**：没有短视频模块也能正常启动，不影响核心发布功能

### P1 应该优化

#### 优化P1-01：添加自动安装Python依赖脚本
新增 `scripts/install-python-deps.js`，自动检测Python和pip，执行pip install -r requirements.txt
**收益**：用户一键安装，无需手动操作

#### 优化P1-02：增强Python错误输出捕获
在spawn调用Python时，捕获stderr并输出到日志，帮助用户排查问题
**收益**：Python执行失败时用户能看到具体错误原因

#### 优化P1-03：完善OpenClaw技能入口
在 skills/wechat-publisher/SKILL.md 补充完整的命令说明，包括 `/wx-setup` `/wx-publish` `/wx-hotspot` `/wx-validate` `/wx-diary`
**收益**：OpenClaw用户可以直接通过命令使用，不需要记住复杂参数

#### 优化P1-04：热点评分优化 - 标题匹配加权
修改评分逻辑：标题匹配关键词分数×2，内容匹配关键词分数×1，提高热点匹配精准度
**收益**：更精准找到用户真正关注的热点

---

### P2 可以优化

#### 优化P2-01：添加发布成功后OpenClaw通知
发布完成后调用OpenClaw notify接口发送通知，用户不用一直盯着终端
**收益**：更好的用户体验

#### 优化P2-02：支持配置封面图水印
在配置文件增加 `cover.watermarkText` 选项，自动在封面添加公众号名称水印
**收益**：品牌露出，提升辨识度

#### 优化P2-03：拆分engine.cjs为多个模块
- hotspot.js - 热点抓取
- scoring.js - 热点评分
- aiwriter.js - AI写作
- wechat-api.js - 微信API
- enhanced.js - 增强发布
**收益**：代码更容易维护，新增功能不用挤在一个文件

#### 优化P2-04：封面配色可配置
配置文件增加 `cover.colors` 数组，用户可以自定义渐变配色
**收益**：满足不同公众号风格需求
