# BOM生成命令

## 功能
基于产品平台采集的数据，自动生成结构化BOM清单：
- 原材料/零部件清单
- 数量、规格、供应商信息
- 成本估算（需对接价格接口）
- 替代料方案（可选）

## 前置条件
1. 已完成产品数据采集（执行`product-data-collect`命令）
2. 已在`config/user-config.json`配置BOM规则：
   ```json
   {
     "bom": {
       "outputFormat": "excel", // 可选：excel/json/markdown
       "includeCost": false,
       "alternativeParts": true,
       "template": "sme-standard" // 中小企业标准模板
     }
   }
   ```
3. 已提供BOM格式规范（必填字段、示例文件）

## 执行流程
1. 读取`data/products/`目录下的产品JSON数据
2. 解析产品规格参数，匹配物料库（默认使用`references/bom-material-library.json`）
3. 按BOM模板生成结构化清单
4. 输出至`output/bom/`目录：
   - Excel格式：`BOM_[产品ID]_[日期].xlsx`
   - JSON格式：`BOM_[产品ID]_[日期].json`
   - Markdown格式：`BOM_[产品ID]_[日期].md`

## BOM输出示例（Markdown）
```markdown
# BOM清单 - 智能工业传感器 ISC-2000

## 基础信息
- 产品ID：PROD-001
- 型号：ISC-2000
- 生成日期：2026-04-29
- 生成人：内容数字员工

## 物料清单

| 序号 | 物料编码 | 物料名称 | 规格型号 | 单位 | 数量 | 供应商 | 备注 |
|------|----------|----------|----------|------|------|--------|------|
| 1    | MAT-001  | 压力敏感芯片 | 量程0-100MPa | 个 | 1 | 芯感科技 | 核心元件 |
| 2    | MAT-002  | 信号放大电路 | 4-20mA输出 | 套 | 1 | 华强电子 | 含PCB |
| 3    | MAT-003  | 防护外壳 | IP67铝合金 | 个 | 1 | 精密压铸 | 定制件 |
| 4    | MAT-004  | 密封件 | 氟橡胶O型圈 | 个 | 2 | 密封科技 | 易耗品 |

## 替代料方案
- 压力敏感芯片：MAT-001可替代为MAT-001A（量程相同，精度更高）
- 信号放大电路：MAT-002可替代为MAT-002B（支持HART协议）

## 备注
- 此BOM适用于小批量生产（<100台）
- 批量采购可联系供应商获取折扣
```

## 待用户提供
1. **BOM格式规范**：必填字段列表、输出模板示例
2. **物料库数据**：现有物料编码、名称、规格、供应商清单（或指定默认库）
3. **成本接口**（可选）：对接价格API，自动计算BOM成本
4. **中小企业特殊要求**：环保认证、RoHS合规等附加字段
