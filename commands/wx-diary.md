# 发布自定义文章到微信公众号

请按以下步骤将用户提供的内容整理并发布：

## 执行步骤

1. **确认内容**：获取用户提供的文章标题和正文内容（支持 Markdown 格式）

2. **内容优化**：
   - 检查标题是否吸引人，若过于平淡建议优化
   - 确保正文包含清晰的段落结构
   - 去掉"明日预告"等限制性内容（避免限制次日写作自由度）
   - 受众定位：面向**中小企业主、个体户和个人**，而不仅限于某个行业

3. **格式转换**：将整理好的 Markdown 内容通过以下命令发布：

```bash
node ${CODEBUDDY_PLUGIN_ROOT}/scripts/engine.js --diary "文章标题" "Markdown正文内容"
```

4. **输出结果**：打印 Media ID 并提示前往 https://mp.weixin.qq.com 草稿箱查看

## 注意事项

- 若配置文件不存在，提示用户先运行 `/wx-setup`
- 正文中的换行用 `\n` 表示
- 文章风格保持轻松幽默+干货满满，避免说教感
